LE TRACÉ DU CIRCUIT EST LE LANGAGE COMMUN DE L'APP

Un pilote pense en virages, pas en écarts-types. Chaque insight se pose donc sur le tracé schématique, à l'endroit exact où il se produit. Le chiffre vit sur la carte ; on ne le lit plus à plat pour imaginer le virage ensuite. Trois maquettes fixent le système — Claude Code l'applique ensuite aux 50 écrans.

Comprendre sa conduite
Voir SES trajectoires sur le tracé, c'est la fonction miroir à son sommet. On montre où ça se passe ; le pilote en tire sa lecture.
Faire évoluer
Le progrès devient géographique : ce virage, session après session, le faisceau se resserre. Visible, jamais dicté.
Coach en bandeau
L'annotation du coach est séparée sous le tracé, ancrée à un point précis, attribuée. Elle n'envahit jamais le miroir du pilote.

Écran A · dispersion spatiale

Le chiffre vit sur le tracé

On ne lit plus « 1,8 m » dans le vide : on voit le faisceau s'évaser au V4 et se confondre au V1. Le pilote comprend instantanément, sans traduire mentalement.

Superposer ses 18 trajectoires réelles sur le tracé schématique : aucune app grand public ne le fait lisiblement. C'est la compréhension de sa conduite, pas un rapport.

POURQUOI C'EST JUSTE

Écran C · soi contre soi spatial

L'écart devient géographique

Deux de SES tours sur le même tracé : ils se collent en S1/S3 et s'écartent visiblement en S2. Le pilote voit OÙ il a changé, pas juste « +1,0 s ».

Le moteur de progression

C'est le cœur de « faire évoluer » : la même vue, session après session, montre le faisceau qui se resserre. Le progrès se constate sur la carte, sans qu'on dicte la trajectoire.

Coach ancré à la zone

Le bandeau pointe la zone S2 entourée — exactement le lieu de l'écart. L'app montre la divergence ; le coach dit qu'on y travaille.

Le système, en une phrase : le tracé schématique dark est le support de tout insight ; le fait s'y inscrit là où il se produit (faisceau, replay coloré, superposition à soi) ; le bandeau coach est séparé sous le tracé, ancré à un point précis, attribué à un tiers agréé.

Cette grille s'applique aux 50 écrans : restitution, détail data, sessions, circuits, garage, communauté. Claude Code reçoit le pattern TrackStage + Facts + CoachBand et le décline. L'app montre, sur le tracé, ce que le pilote ne voyait pas seul. Elle ne dirige jamais. La piste — et les décisions — restent à lui.
